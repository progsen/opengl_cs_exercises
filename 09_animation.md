
## Oefening 1
**Doel:** een animatie kunnen maken


- lees:
```
als je beide frames kan weergeven dan weet je hoe je bij andere frames met de UV kan komen

- als je animeert heeft je animatie vaak een snelheid, bv per 0.5 seconde naar het volgende frame
- je moet dan zelf een timer bijhouden waar je de time van het frame bij optelt
    - als deze groter is dan de tijd per frame ga je naar de volgende frame
    - dan zet je de UV coordinate van het frame

- hoe kom je aan je UV?
    - die kan je van te voren in een 2 dimensionale array zetten
    - of uitrekenen als je plaatje met je animation frames goed opgezet is
```
